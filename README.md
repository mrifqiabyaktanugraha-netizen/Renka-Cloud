# Reincity Cloud — Static Demo

Proyek statis demo untuk Reincity Cloud: landing page modern dengan tema biru laut dan neon effect.

Isi proyek:
- `index.html` — Halaman utama (hero, layanan, harga, kontak)
- `services.html` — Rincian layanan
- `docs.html` — Contoh dokumentasi API
- `css/custom.css` — Styling tambahan (neon, animasi)
- `js/main.js` — Interaktivitas ringan
- `assets/logo.svg` — Logo placeholder

Cara menjalankan:
1. Buka `index.html` di browser (double-click atau "Open with Live Server" jika Anda menggunakan extension).
2. Karena Tailwind dipakai melalui CDN, tidak perlu instalasi tambahan.

Catatan:
- Konten berbahasa Indonesia untuk demo.
- Logo dan ilustrasi adalah placeholder; ganti sesuai kebutuhan.

Pengembangan lebih lanjut (opsional):
- Tambahkan Tailwind JIT/builder saat ingin meng-custom utility.
- Hubungkan API backend dan autentikasi.

Lisensi: bebas digunakan untuk demo/percobaan.
